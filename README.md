## Development

### Install Dependencies

```bash
$ npm install
```

### Start Development Server

Changes to project will trigger build

```bash
$ npm run build:watch
```

### Run All Tests

```bash
$ npm test
```

### Debug Browser Tests

```bash
$ npm test:wtr:watch
```

### Lint

```bash
$ npm run lint
```

### Build All

```bash
$ npm run build
```

### Build Chrome

```bash
$ npm run build:chrome
```

### Build Safari

```bash
$ npm run build:safari
```
